Lazy Spy Chrome Extension - Download Package
============================================

This package contains the Lazy Spy Chrome extension files.

INSTALLATION INSTRUCTIONS:
1. Extract all files from this zip
2. Open Chrome and go to chrome://extensions/
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the extracted folder
5. The Lazy Spy extension will now be installed

FEATURES:
- Visual analysis of image loading strategies
- Color-coded overlays for different loading types
- Performance insights and recommendations
- Support for multiple lazy loading libraries
- Export detailed reports

For more information, visit: https://lazyspy.com

Note: This is a development version. The official version will be available on the Chrome Web Store soon.

What Lazy Spy Does

Comprehensive image performance analysis with actionable insights

Visual Analysis

Color-coded overlays show lazy loaded (blue), eager loaded (yellow), preloaded (teal), and problematic images (red)

Smart Detection

Automatically identifies native lazy loading, JavaScript libraries, and custom implementations

Performance Insights

LCP detection, fetch priority analysis, CDN identification, and Core Web Vitals impact

Actionable Recommendations

Missing dimensions calculator, format conversion suggestions, preload opportunities

Library Support

Detects LazySizes, LoZad, React LazyLoad, Vue LazyLoad, Intersection Observer, and custom implementations

Export Reports

Generate detailed CSV reports for team analysis and performance tracking
